2. Release文件夹
	exe文件目录， work.exe为openGL项目的可执行文件，双击运行
	
	操作说明：
		1）鼠标操作：左击————恢复视角：例如转动视角之后，可以恢复和车辆的相对观察位置
		2）键盘操作：
			车辆操作：
				1）pageUP: 加速;   pageDown:减速
				2）pageLeft:左转； paegRight:右转（5°）
				3）Space(空格)：启动/停止车辆
				4）R(or r):		车辆撞向路边之后翻车，R键可以复位
			摄像机操作（视角）：
				1）W：向上拉升
				2）S：向下降低
				3）A：向左移动
				4）D：向右移动
				5）N：向前移动
				6）M：向后移动
	
	操作实例：space-->启动， pageUP-->加速， pageLeft:左转
	
	注意：建议使用独显运行，否则容易出现卡顿、掉帧现象!!!!
